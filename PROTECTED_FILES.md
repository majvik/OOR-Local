# Защищенные файлы

## Файлы с ограниченным доступом

### src/js/slider.js
- **Статус**: Защищен от редактирования
- **Права доступа**: 444 (только чтение)
- **Причина**: Критический функционал слайдера, не должен изменяться
- **Дата защиты**: 2025-09-21

## Команды для управления защитой

### Установить защиту (только чтение):
```bash
chmod 444 src/js/slider.js
```

### Временно снять защиту (для редактирования):
```bash
chmod 644 src/js/slider.js
```

### Восстановить защиту после редактирования:
```bash
chmod 444 src/js/slider.js
```

## Важно
- Перед любыми изменениями в защищенных файлах снимайте защиту
- После редактирования обязательно восстанавливайте защиту
- Не коммитьте изменения в защищенные файлы без веской причины
